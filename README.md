# Анализ покупательского поведения

## Описание проекта
В проекте проводится анализ данных о покупках клиентов интернет-магазина
с целью оценки влияния подписки на средний чек и поведение покупателей.

## Используемые инструменты
- Python
- pandas
- Jupyter Notebook

## Основные бизнес-вопросы
- Отличается ли средний чек у клиентов с подпиской и без подписки?
- Влияет ли подписка на покупательское поведение?

## Основные выводы
- Средний чек у подписчиков и неподписчиков практически не отличается.
- Более высокая суммарная выручка у клиентов без подписки объясняется
  большим числом таких клиентов в датасете.
- Анализ частоты повторных покупок ограничен структурой данных
  (одна покупка на клиента).

## Ограничения
- Датасет содержит одну покупку на одного клиента.
- Отсутствует временная динамика заказов.

## Структура проекта
- `notebooks/` — ноутбук с анализом
- `data/` — исходные данные
